VIEW DESCRIPTION AND KEY COMPONENTS

Description

The Manage view provides access to all the phases, activities and tasks associated with the Manage focus area (project management and program management), including supplemental content and reference files. This view is presented with an emphasis on the Manage activities with the tasks and work products organized in a work breakdown structure by phase and activity.
