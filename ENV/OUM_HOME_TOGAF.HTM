VIEW DESCRIPTION AND KEY COMPONENTS

Description

The TOGAF view provides support for The Open Group Architecture Framework (TOGAF). TOGAF® is an industry standard architecture framework that can be used freely by any organization wishing to develop an Information Systems Architecture for use within that organization.
