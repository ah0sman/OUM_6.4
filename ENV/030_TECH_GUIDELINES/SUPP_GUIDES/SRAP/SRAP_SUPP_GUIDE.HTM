SERVICE-ORIENTED ARCHITECTURE (SOA) REFERENCE ARCHITECTURE PLANNING SUPPLEMENTAL GUIDE

The materials presented in this technical guideline provide specific supplemental information into OUM for the purposes of SOA Reference Architecture Planning. The SOA Reference Architecture Planning Supplemental Guide contains the following components:

SOA Reference Architecture Planning Overview

This component provides an overview of the concepts of SOA Reference Architecture Planning.

SOA Reference Architecture Planning Supplemental Task Guidelines

The Supplemental Task Guidelines should be used in conjunction with the standard OUM tasks in order to accomplish SOA Reference Architecture Planning work at the detail task level.

SOA Reference Architecture Planning Technique/Tool Guidelines

The Technique/Tool Guidelines provide Information on how to apply SOA Reference Architecture Planning techniques/tools through the tasks in OUM.

PDF View

You may also prefer to access this information through the SOA Reference Architecture Planning Supplemental Guide (PDF View) located in the Supplemental Guides section of the Supplemental Guidance page.

PURPOSE

The purpose of this document is to provide an overview of a SOA Reference Architecture Planning engagement, and to provide implementation teams with supplemental guidance for the successful planning and execution of SOA Reference Architecture Planning engagements.

Back to Top

AUDIENCE

This document is intended for practitioners who will be delivering SOA Reference Architecture Planning engagements based on the OUM (Oracle Unified Method) approach.

Back to Top

HOW TO USE THIS DOCUMENT

This document contains supplemental guidance for planning and executing SOA Reference Architecture Planning engagements and should be used in conjunction with and not instead of the OUM base method guidance. In many cases the work that needs to be done for the SOA Reference Architecture Planning process is fully contained within the OUM base method task and technique files, therefore, no task supplement is needed for that task.
