Manage View

Description

The Manage view provides access to all the phases, activities and tasks associated with the Manage focus area (project management and program management), including supplemental content and reference files. This view is presented with an emphasis on the Manage activities and the tasks and work products are organized in a work breakdown structure by phase and activity.

Envision View

Description

The Envision view provides access to all the phases, activities and tasks associated with the Envision focus area, including supplemental content and reference files. Envision concerns IT related issues at the enterprise level from an organizational, governance and architectural point of view. If not already started, Envision can be initiated at any time and should be continued for as long as the enterprise exists. Envision spans the whole project lifecycle of current and future IT projects. This view supports pre-sales, and Enterprise Architect disciplines.
