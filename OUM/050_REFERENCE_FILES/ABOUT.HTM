About

Description

The Oracle® Unified Method (OUM) is Oracle's full lifecycle method for deploying Oracle-based business solutions. OUM includes the four focus areas: Manage, Envision, Implement and Operate. You can tailor OUM to support your specific project situation. With its ready-made templates, guidelines, and tailored work breakdown structure, OUM provides the programmatic tools you need to manage the risks associated with your Oracle-based projects.

De-Installation

Windows 7: Removing OUM 6.4.0 through 4.3.0:

  1. Click "Start" > "Control Panel" > "Programs or Features ".
  2. Locate "Oracle OUM (Oracle Unified Method) 6.X.0", "Oracle OUM (Oracle Unified Method) 5.X.0" or "Oracle OUM (Oracle Unified Method) 4.X.0" where "x" represents the second digit in the release number.
  3. Right-click and select "Change".
  4. Select the "Next" button.
  5. Select the "Remove" radio button.
  6. Select the "Next" button.
  7. Select the "Remove" button.

Windows XP: Removing OUM 6.4.0 through 4.3.0:

  1. Click "Start" > "Control Panel" > "Add or Remove Programs".
  2. Locate "Oracle OUM (Oracle Unified Method) 6.X.0", "Oracle OUM (Oracle Unified Method) 5.X.0" or "Oracle OUM (Oracle Unified Method) 4.X.0" where "x" represents the second digit in the release number.
  3. Select "Change" button.
  4. Select the "Next" button.
  5. Select the "Remove" radio button.
  6. Select the "Next" button.
  7. Select the "Remove" button.

Removing OUM 4.2.1:

This is a two step process:

  1. Delete the OUM 4.2.1 Program Icon.
    • Click "Start" > "Programs" > "Oracle Methods".
    • Right-mouse click over "Oracle OUM (Oracle Unified Method) 4.2.1"
    • Select "Delete" from listed options.
  2. Delete the OUM 4.2.1 Method-specific Components.
    • Open "Windows Explorer".
    • Navigate to the "C:\METHOD\OM40\OUM_4.2" directory.
    • Delete the "OUM_4.2" directory. Important: Do not delete parent directories of OUM_4.2 (e.g., C:\METHOD).
