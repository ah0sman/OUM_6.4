Internet Explorer: Configuring IE to Open Templates in the Appropriate MS Office Program

Unlike Netscape, the default settings for Internet Explorer 6 and earlier versions cause templates to be displayed within the browser when they are selected via the links from the OUM method pack. To configure Internet Explorer to open templates in the appropriate Office program:

  • Open "My Computer".
  • Select "Tools" > "Folder Options".
  • Select the "File Types" tab.
  • From the Registered files types list, select the specific Office document type (for example MS Word Document)
  • Select the Advanced button.
  • In the Edit File Type dialog box, clear the "Browse in same window" checkbox.
  • Click "OK".

OUM templates are Read-Only. Use the "File"+"Save As" menu option to save a copy of the template to a directory on your computer (other than under C:\METHOD). You can then open the saved file, using MS Word, for editing.

Note: See the Template User's Guide, accessible from the Resources page, for more information on working with templates.

MS Word: Generating Work Products from OUM Templates

To generate a work product from an OUM template:

  • Locate and select the appropriate template.
  • Save the template to another directory on your computer using “Save As...”.
  • Close MS Word.
  • Navigate to the saved file and open it.
