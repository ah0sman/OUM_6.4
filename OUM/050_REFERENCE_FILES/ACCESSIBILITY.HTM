Accessibility and OUM

Oracle's Global Methods team has taken steps toward making OUM more accessible. If you have problems accessing something in the OUM Method Pack or have a suggestion for improving the accessibility of the OUM Method Pack, please contact Oracle's Global Methods team.

STANDARDS

OUM uses HTML 4.01 for structure and cascading style sheets (.css) for design. As such, the OUM Method Pack should display in any standard-compliant browser and should be accessible from different devices, such as screen readers and mobile devices.

MENU NAVIGATION

Mouse and/or keyboard shortcuts can be used to navigate through the OUM Method Pack. When using keyboard shortcuts, it is best to use Mozilla Firefox. Other browsers do not provide the same level of functionality.

OUM Method Pack keyboard shortcuts include:

  • "Shift + Alt + s" to access the Select a View drop-down menu on the OUM Home page .
  • "Shift + Alt + m" to access the Method Navigation drop-down menu available on most OUM pages.
  • "Shift + Alt + c" to access the Current Page Navigation drop-down menu available on most pages.

Use the up and down arrow keys to move through the options on the drop-down menus. Press Enter or Tab to go to the desired option on the drop-down menu.

Select the Oracle Unified Method logo, located on the top left corner of every page, to go to the OUM Home page.

OTHER

The OUM Method Pack provides the following accessibility features:

  • Relative sizing has been used so that you can use the browser's controls to change the font size.
  • All images have alternate text.
  • All tables have summary descriptions and scope attributes.
