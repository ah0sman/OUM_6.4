Adobe Reader: Links in Adobe Acrobat (PDF) Views Do Not Work

With OUM 6.3.1, OUM is not longer available as a desktop installable Method Pack. As such, any navigational links in the PDF views will not work. Use the bookmarks to navigate through the PDF views and log onto the Hosted version of OUM to take advantage of external links.

Internet Explorer: Popup Blocker May Prevent Links and Drop-Down Menus from Working Properly

The Popup Blocker included with Windows XP versions of Internet Explorer may impede the proper functioning of links and drop-down menu items included in the OUM method pack.  If a link or drop-down menu item fails to work properly on your Windows XP machine, check for the popup toolbar at the top of the page.  Click the popup toolbar to remove the block and allow the link or drop-down menu item to function properly.  This is only a temporary fix.

If you choose to inactivate the popup blocker,

  • Launch Internet Explorer.
  • Select "Tools" > "Internet Options".
  • Select the "Advanced" tab.
  • Scroll down to the "Security Settings".
  • Select the check box next to the "Allow Active Content to Run in Files on my Computer" item. This disables the popup blocker. Note: You should consider reactivating the popup blocker after using the OUM method pack, and before browsing the internet again.

Internet Explorer: "To Help Protect Your Security, Internet Explorer Has Restricted this File From Showing Active Content That Could Access Your Computer. Click Here for Options..." Message

To remove this message, you need to enable or allow active content.

When using Windows XP and Internet Explorer,

  • Click on the message.
  • Select "Allow Blocked Content...".
  • From the security warning, select "Yes".

When using Windows 7 and Internet Explorer 8, to eliminate the error messages:

  • Open Internet Explorer.
  • Select "Tools".
  • Select "Internet Options".
  • Select the "Advanced" tab.
  • Scroll to the "Security" section.
  • Select the checkbox for "Allow active content to run in files on My Computer".
  • Select "OK" button.

The change takes affect after you restart Internet Explorer.

Mozilla Firefox: Links from Current Page Navigation Do Not Go To Destination

The Diagram Navigation and top Guidelines Area links from the Current Page Navigation do not go to the destination when using Mozilla Firefox. They only expand the window for the selection. The workaround is to scroll down to the expanded window or use Internet Explorer.

MS Word: Read-Only Warning

MS Word templates are defined for read-only access in order to prevent you from accidentally making changes to the base template files. Use the "File"+"Save As" option to save a copy of the template to a directory on your computer other than under C:\METHOD. Note: You will see an MS Word read-only warning, if you choose to "Save" the MS Word template rather than "Save As".
