Navigate OUM

OUM contains the complete set of material associated with the method. The Guidelines are provided in HTML for easy, on-line navigation and PDF for easy printing. Use the PDF View button in the header of any HTML page to access a PDF view of the core method. The templates are provided in various formats, including MS Office. Starting with this release, OUM is only available in a hosted format.

VIEW PAGES

The views are predefined subsets of the method material. From the categories listed in the Access Method By drop-down, select a view from your area of interest or that is applicable to your engagement. The view pages include links to detailed information about OUM's phases, processes and tasks, as well as links to templates and key reference materials pertinent to a particular view. The View Catalog provides a list of views and descriptions. Following is a description of the format and features of the view pages.

The top section of the view page provides a description of the view and links to key components that are relevant to that view.

The view pages provide several types of navigation:

  • Diagram Navigation
  • Guidelines Navigation
  • Method Navigation Drop-down Menu
  • Current Page Navigation Drop-down Menu

The page is dynamic allowing you to expand or collapse sections so that you can view the content you are most interested in at any point in time.

Diagram Navigation

The Diagram Navigation section provides a diagram, which depicts either OUM's phases and processes or view-appropriate alternative areas.

If you click on one of the bars representing the intersection of a phase and process or one of the areas, you'll be directed to a table that lists all of the tasks for that process or area, organized by phase. The table is located in the lower section of the view page.

  • To expand the diagram, click on the icon.
  • To collapse the diagram, click on the icon.

Guidelines Navigation

Immediately below the diagram is a table that provides links to the phase overviews, process overviews, tasks and templates, and references.

  • To expand the table, click on the icon.
  • To collapse the table, click on the icon.

The lower portion of the view page provides a table that lists all of the tasks for each process, organized by phase. This is the same table you will access if you select a phase/process intersection from the diagram (located at the top of the view page).

  • To expand a section of the table, click on the icon.
  • To expand the entire table, click on the "expand all" link.
  • To collapse a section of the table, click on the icon.
  • To collapse the entire table, click on the "collapse all" link.

The tasks and templates displayed in the table are specific to a particular view of OUM. All of the tasks, and their corresponding templates, are listed by process.

  • To access a task description, click on the task name (from the Task Overview column).
  • To access a template, click on the template name (from the Template column).
    Note: The appropriate application (for example, MS Word) will launch and the template will be displayed in that application.

Method Navigation Drop-Down Menu

The Method Navigation drop-down menu allows you to navigate through the method content.

Links from the drop-down menus do not invoke a new page. Use the "Back" button in your browser to return to a previous page.

Current Page Navigation Drop-Down Menu

The Current Page Navigation drop-down menu allows you to navigate through the method content on a particular page.

Use the "Back to Top" links to return to the top of the page.

RESOURCES PAGE

The Resources page provides links to valuable OUM content. You can always come directly to this page by using the Resources button located in the header of most OUM pages. Among the links included on this page are links to the Project Roles and Definitions and Terms.
