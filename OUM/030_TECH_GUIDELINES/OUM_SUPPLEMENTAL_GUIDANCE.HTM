  • Oracle Method Template Engine (OMTE) Template User's Guide (PDF) - Use this link to access the RETIRED OMTE Template User's Guide that provides detailed instructions for using work product templates or work products created from templates prior to OUM 6.3. With Release 6.3, the Oracle Method Template Engine was retired. This User’s Guide is also retired. It is being made available as a reference document for use when working with work products that had been created with versions of OUM prior to Release 6.3. The Oracle Method Template Engine is no longer being supported or enhanced. Templates in OUM Release 6.3 no longer use the Oracle Method Template Engine. Refer to the Template User’s Guide above for instructions on working with templates from OUM Release 6.3.
