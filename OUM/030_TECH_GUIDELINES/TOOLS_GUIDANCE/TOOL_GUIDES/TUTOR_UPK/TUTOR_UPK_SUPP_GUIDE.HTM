ORACLE USER PRODUCTIVITY KIT PROFESSIONAL (UPK PRO) AND ORACLE TUTOR SUPPLEMENTAL GUIDE

The materials presented in this tool guideline provide supplemental guidance for using Oracle® User Productivity Kit Professional (UPK Pro) and Oracle® Tutor in conjunction with OUM. Oracle Tutor provides users with tools to develop relevant process documentation and learning materials. Oracle UPK provides tools for annotated transaction simulations depicting proper field-by-field execution of key system processes. UPK Pro Knowledge Center provides an easy-to-search knowledge portal that provides stakeholders, project managers and employees with access to a central repository of all project assets. Used with available baseline content, the materials developed by these tools are process and role-based, easy to update, cost-effective and developed in conjunction with the implementation to accurately depict the future state model. Implementation documentation developed with UPK Pro and Tutor can be leveraged and used to create training content to enable end users to be productive at go-live.

PURPOSE

This document contains OUM supplemental task guidance for using User Productivity Kit Professional (UPK Pro) and Oracle Tutor in conjunction with OUM. UPK Pro and Tutor are complementary Oracle tools that are used to develop and maintain content that can be used throughout each phase of the enterprise application implementation lifecycle from blueprinting/design/testing to go-live and maintenance/support. UPK Pro and Tutor can be used with all of Oracle's business application brands as well as non-Oracle business applications. UPK Pro and Tutor allow project teams to streamline the creation of business process documentation, test scripts, interactive simulations, job aids, presentations, courseware and in-application performance support reflecting the specific business practices and applications environment. For creating the above content types, use Tutor for business process documentation and UPK Pro for everything else.
