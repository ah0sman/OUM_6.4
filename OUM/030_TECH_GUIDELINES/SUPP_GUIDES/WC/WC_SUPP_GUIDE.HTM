WEBCENTER SUPPLEMENTAL GUIDE

The materials presented in this technical guideline provide specific supplemental information into OUM for the purposes of WebCenter delivery work.The WebCenter Supplemental Guide contains the following components:

WebCenter Overview

This Overview provides a summary of the WebCenter platform:

  • Oracle WebCenter Sites
  • Oracle WebCenter Portal
  • Oracle WebCenter Content
  • Oracle WebCenter Social

WebCenter Supplemental Activity Guidelines

The Supplemental Activity Guidelines should be used in conjunction with the standard OUM activity guidelines in order to accomplish WebCenter project delivery work at the activity level.

WebCenter Supplemental Task Guidelines - Inception Phase

The Supplemental Task Guidelines for the Inception phase should be used in conjunction with the standard OUM Task guidelines in order to accomplish WebCenter project delivery work at the task level.

WebCenter Supplemental Task Guidelines - Elaboration Phase

The Supplemental Task Guidelines for the Elaboration phase should be used in conjunction with the standard OUM Task guidelines in order to accomplish WebCenter project delivery work at the task level.

WebCenter Supplemental Task Guidelines - Construction Phase

The Supplemental Task Guidelines for the Construction phase should be used in conjunction with the standard OUM Task guidelines in order to accomplish WebCenter project delivery work at the task level.

WebCenter Supplemental Task Guidelines - Transition Phase

The Supplemental Task Guidelines for the Transition phase should be used in conjunction with the standard OUM Task guidelines in order to accomplish WebCenter project delivery work at the task level.

WebCenter Supplemental Task Guidelines - Production Phase

The Supplemental Task Guidelines for the Production phase should be used in conjunction with the standard OUM Task guidelines in order to accomplish WebCenter project delivery work at the task level.

WebCenter Definitions and Terms

The WebCenter Definition and Terms should be used in conjunction with the standard OUM Definitions and Terms in order to understand terms referenced in this Supplemental Guide.

WebCenter Projects and OUM

WebCenter projects require only a subset of the full list of activities and tasks included within OUM.

When using OUM on your project, it is important to let the method serve you, that is, only select the appropriate activities and tasks that you need.

WebCenter Project Workplan (MPP)

OUM provides a generic Project Workplan, in Microsoft Project format, that can be quickly tailored for WebCenter projects.

PURPOSE

The purpose of this document is to provide an overview of WebCenter use for providing WebCenter solutions, and to provide implementation teams with supplemental guidance for the successful planning and execution of WebCenter implementations.

AUDIENCE

This document is intended for practitioners who will be designing and implementing WebCenter solutions based on the OUM (Oracle Unified Method) approach.
