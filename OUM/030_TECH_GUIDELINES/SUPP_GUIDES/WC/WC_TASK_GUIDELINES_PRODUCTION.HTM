E.ACT.MPSP MANAGE PRODUCTION SYSTEM PERFORMANCE GUIDANCE

PT.120 Conduct Production Performance Management

OVERVIEW

Implement the Performance Management activities to the Production Environment in conjunction with the implementation of the WebCenter system.

Monitoring, metric collection and review is an ongoing process. This may not be within the scope of your project or may covered by a group outside of the project team, after the system goes into production.

E.ACT.EPS EVALUATE PRODUCTION SYSTEM GUIDANCE

PS.010 Audit System

OVERVIEW

Request an independent quality auditor to review the WebCenter system. This task should be performed after cutover, when the WebCenter system has reached a steady state.

APPROACH
  • Review objectives and top-level requirements.
  • Review Security and Control Strategy.
  • Review technical and system operational documentation.
  • Examine the System in Production.

E.ACT.RPP RESOLVE PRODUCTION PROBLEMS GUIDANCE

PS.050 Analyze Problems

OVERVIEW

Analyze each open problem report from the Problem Log (PS.060) and determine which open problems represent faults in the WebCenter system. Determine the changes that are necessary to resolve the set of faults.

Perform this task in parallel with the task, Monitor and Respond to System Problems (PS.060), post Production, if this is within the scope of the project.

PS.060 Monitor and Respond to System Problems

OVERVIEW

Monitor and respond to problems and issues throughout the post-Production support period. It is recommended to also monitor Service Level Agreements (SLAs). This is an ongoing task that may not be full-time.

APPROACH
  • Record and acknowledge problem reports from users and administrators.
  • Compile and maintain the Problem Log.
  • Monitor related SLAs.
