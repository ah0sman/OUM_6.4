Description

The Enterprise Security view provides access to the material required for using OUM on Enterprise Security projects. The supplemental guidance is provided to assist in tailoring project plans, based on the security domains defined by the International Information Systems Security Certification Consortium (ISC)2™. You may also use this view to understand the work done by security architects during Enterprise Security implementation projects.
