This view provides a series of example work products, in OUM format, based on the SkiNow case study used in the OUM training courses. This is not a complete set of work products that might be produced on every OUM-based project, nor would you be expected to complete all of these work products for every project. You would produce just enough documentation to satisfy the needs of your project.
